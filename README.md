# Trabalho-III-IA

Implementação do terceiro trabalho da disciplina de IA

## 1 - Implementação de árvore de decisão manual

Sistema de recomendação de hobbies utilizando uma árvore de decisão manual implementada em Python.

### Estrutura do Projeto

- `perguntas.json`: Contém a árvore de decisão com perguntas e resultados
- `arvore_decisao.py`: Implementação que navega pela árvore através de perguntas ao usuário

### Como Executar

```bash
python arvore_decisao.py
```

O sistema fará uma série de perguntas (sim/não) e, com base nas respostas, recomendará hobbies adequados ao perfil do usuário.
